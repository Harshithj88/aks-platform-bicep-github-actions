# Resume Bullets

- Built a production-style Azure Kubernetes Service platform using Bicep and GitHub Actions, provisioning AKS, ACR, Key Vault, Log Analytics, managed identity, and environment-specific configurations.

- Implemented secure GitHub Actions authentication to Azure using OIDC and Microsoft Entra workload identity federation, eliminating long-lived service principal secrets.

- Designed modular Bicep templates for reusable cloud infrastructure deployment across dev and prod environments.

- Automated AKS infrastructure validation and deployment using GitHub Actions with environment-based workflow inputs and deployment governance.

- Configured AKS integration with Azure Container Registry and Log Analytics to support container image pulls, monitoring, and operational visibility.
